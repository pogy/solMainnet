# Backpack Exchange API Client

This is a reference implementation of the Backpack Exchange API

## Dependencies

- NodeJS
- NPM

## Installation

```sh
npm install
```

## Example

- Login to Backpack Exchange and navigate to Portfolio -> Settings -> API Keys
- Generate a new API key and update index.js
- Ensure you have at least $5 USDC in your account
- Run the example:

```sh
node index.js
```
